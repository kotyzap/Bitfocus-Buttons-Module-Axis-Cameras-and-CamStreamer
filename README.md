# Bitfocus-Buttons-Module-Axis-Cameras-and-CamStreamer
Fifth member of the *Stream Deck on Axis* family — a native **Bitfocus Companion** module. Unlike the Generic-HTTP button page (`../companion-for-axis/`), this gives real dropdown actions populated from the camera, live-state **feedbacks**, and proper **digest auth** (so it works on default Axis configs).
